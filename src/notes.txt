^ HOW TO RICE
b = ceil(ln()/ln(2)) = 4
M = [2^b,2^(b-1)]
cutoff = 2^b - M

... |b bits of data N| ... | ...
q = N/M
r = N%M

q stored as: <1 q times>0
r stored as: binary r (removing preceeding 0 if possible)
code = 0bM|q0|r0|q1|r1...qn|rn

EXAMPLE:
data = 0b100111010010110101001010101
b = 4
M = 2^(b-1)+1 = 2^3 = 8

1)	N = 1001 = 9
	q = 9/8 = 1
	r = 9%8 = 1
	q|r = 10|000
2) N = 1101 = 13
	q = 13/9 = 1
	r = 13%9 = 4
	q|r = 10|100

M = 8
N = [0,15] = [0000,1111] (b)
q = [0,1] = [0,1]
r = [0,7] = [000,111]
